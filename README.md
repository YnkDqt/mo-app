# Mo — Suivi de cycle · Symptothermie

Application React de suivi de cycle menstruel basée sur la méthode symptothermique.

## Stack
- React 18 + Vite
- Recharts pour les graphiques
- Zéro framework CSS

## Développement local

```bash
npm install
npm run dev
```

## Déploiement
Vercel via GitHub — push sur `main` → déploiement automatique.

## Données
Utilise le bouton "💾 Données" dans la sidebar pour exporter/importer tes données en JSON.  
Charge `demo-data.json` (dans `public/`) pour tester avec les données historiques.
